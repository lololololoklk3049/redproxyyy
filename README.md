# 🔴 Red Proxy

A powerful, censorship-bypassing web proxy with a stunning red and black glowing particle theme. Built with Scramjet and Epoxy for maximum performance and compatibility.

## ✨ Features

- 🚀 **Scramjet Core** - Latest proxy technology for maximum site compatibility
- 🌐 **Epoxy/Wisp Support** - High-performance WebSocket transport
- 🎨 **Red & Black Theme** - Glowing particle effects with Google-like simplicity
- 🔒 **Privacy Focused** - No logs, secure browsing
- 📱 **Responsive Design** - Works on all devices
- ⚡ **Quick Links** - One-click access to popular sites
- ⌨️ **Keyboard Shortcuts** - Ctrl/Cmd+K to search, Esc to clear
